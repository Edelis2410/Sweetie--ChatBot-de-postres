# Sweetie - Chatbot de Postres

**Sweetie** es un asistente virtual con personalidad de chef pastelera, lista para ayudarte con recetas, recomendaciones y todo lo relacionado con el mundo de la repostería. Construido con HTML, CSS y JavaScript puro, se conecta localmente con **Ollama** para generar respuestas inteligentes y con formato.
